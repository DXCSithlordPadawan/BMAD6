Upload a Markdown (.md) file to add a new template or agent to the library. The file will be parsed automatically — the first # Heading becomes the template name and each ## Heading becomes a section. If a template with the same name already exists it will be overwritten.

Markdown Format

Structure your .md file like this:

---
name: My Agent Name
is_agent: true
groups:
  - Planning
  - Development
---

# My Agent Name

## Persona
Describe the agent persona here.

## Workflow
List the steps the agent should follow.

## Constraints
Any hard limits or rules for the agent.

Use is_agent: false for document templates (e.g. Project Brief, ADR).
